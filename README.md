# 🛒 SkyMart - Modern React E-Commerce Website

SkyMart is a modern and responsive e-commerce web application built using React. It provides users with a seamless shopping experience featuring authentication, product browsing, category filtering, shopping cart management, and protected routes.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/skymart.git
```

Move into the project

```bash
cd skymart
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

# 🔒 Authentication Flow

- User registers with email and password.
- Email duplication is prevented.
- User information is stored in Local Storage.
- Login validates credentials.
- Protected Routes prevent unauthorized access.
- Public Routes prevent logged-in users from visiting login/signup pages.
